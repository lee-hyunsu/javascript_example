01. 배열 정의
    (배열 정의 및 배열 데이터의 값을 가져오고 싶을 때)

02. 배열의 길이 확인
    (배열 데이터 요소의 수를 확인 및 특정 JSON데이터 요소의 수를 확인하고 싶을 때)

03. 배열 요소 다루기_1
    (배열 요소의 데이터를 처리하고 싶을 때)

04. 배열 요소 다루기_2
    (배열의 루프 처리 중 요소의 인덱스가 불필요할 때)

05. 배열 요소 다루기_3
    (배열의 루프 처리 중 요소의 인덱스가 필요할 때)

06. 배열 요소 추가하기
    (배열에 요소 추가 및 요소를 배열의 처음 혹은 마지막 부분에 추가하고 싶을 때)

07. 배열 요소 삭제하기
    (배열의 요소를 삭제하고 싶을 때)

08. 배열 요소 부분 변환하기
    (배열 내 요소를 다른 요소로 변환하고 싶을 때)

09. 배열 결합하기
    (여러 개의 배열을 하나로 결합하고 싶을 때)

10. 배열 요소 결합하여 문자열 만들기
    (배열 내 요소의 문자 데이터를 결합하곳 싶을 때)

11. 배열 요소 검색하기
    (배열 데이터의 특정 요소를 확인하고 싶을 때)

12. 조건을 만족하는 배열 요소 가져오기
    (유저 정보가 담긴 배열에서 ID를 기준으로 정보를 가져오고 싶을 떄)

13. 배열 요소 역순 정렬하기
    (배열 요소를 역순으로 정렬하고 싶을 때)

14. 배열 요소 정렬 방법 지정하기
    (배열 요소를 오름차순/내림차순으로 정렬하고 싶을 때)

15. 객체를 포함하는 배열 정렬하기
    (배열 요소를 오름차순/내림차순으로 정렬하고 싶을 떄)

16. 배열 요소 알파벳순 정렬하기
    (대소문자 구분 없이 알파벳순으로 정렬하고 싶을 때)

17. 배열 요소를 추출하여 새 배열 만들기
    (요소를 추출하여 새로운 배열을 만들고 싶을 때,
     배열 요소 전체에 대한 처리 작업을 하고 싶을 때,
     ID와 이름을 가지는 객체 배열에서 ID만 가지는 배열을 새로 만들고 싶을 때)

18. 조건을 만족하는 배열 요소 추출하여 새 배열 만들기
    (유저 정보 배열에서 18세 이상인 유저의 정보만을 가져와 배열을 생성하고 싶을 때)

19. 배열 요소 하나로 정리하기
    (배열의 요소를 계산하여 하나의 값으로 만들고 싶을 때)

20. 유사 배열 객체를 배열로 변환하기
    (문자열과 유사 배열(ArrayLike)을 배열로 변환하고 싶을 때,
     반복 가능(Iterable)한 객체를 배열로 변환하고 싶을 때)

21. 요소에 개별 변수에 분할 대입하기
    (배열의 요소를 변경하고 싶을 때)

22. 배열 섞기(셔플)
    (게임에서 요소의 값을 섞을 때)

23. 다양한 데이터 타입을 가진 객체 사용하기
    (다양한 데이터를 하나의 객체로 만들고 싶을 ㄸㅐ
     연과 배열을 사용하고 싶을 때)

24. 객체 선언, 수정, 확인하기
    (객체의 속성을 변경하고 시을 때)

25. 객체 복사하기
    (데이터를 복사하고 싶을 떄)
